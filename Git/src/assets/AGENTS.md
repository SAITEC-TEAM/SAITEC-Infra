# Agent Entry

本仓库的 AI 协作主规范在 `.saitec/AI_COLLABORATION.md`，Git 协作技能清单在 `.saitec/Skills.md`。

执行任务前请先：
1. 阅读相关代码、配置和文档。
2. 如任务涉及分支、提交、PR 或交付说明，再阅读 `.saitec/Skills.md`。
3. 明确目标、约束和验收标准。
4. 仅做最小必要改动，避免无依据重构。

输出结果时请明确：
- 修改内容与原因
- 验证方式与结果
- 残余风险、假设或未完成项

如与 `Skills.md` 或工具专属配置冲突，以 `.saitec/AI_COLLABORATION.md` 为准。
